# CDEWS-IAFS
## Cognitive Drift Early Warning System

*Version:* 5.0 | *License:* MIT | *Python:* 3.9+

## Description
CDEWS-IAFS is an independent monitoring layer that evaluates the structural stability of AI-generated text in real time. No API required. Fully local. Deterministic results.
